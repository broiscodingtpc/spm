# Episodes – Web playback

Browsers on Windows (Chrome, Firefox, Edge) often **do not play .MOV (QuickTime)** files.

To have the videos play on the site:

1. Convert each `.MOV` to **MP4 (H.264)**.
2. Save the MP4 files in this folder with the **exact** names:
   - `introtrailer.mp4`
   - `pilotepisode.mp4`
   - `episode1.mp4`
   - `episode2.mp4`

The site will use the MP4 version when available. You can use free tools like HandBrake or FFmpeg to convert MOV to MP4.
